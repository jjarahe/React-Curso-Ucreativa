Fundamentos

  1. Se basa en un contenedor
  2. Contiene Flex items
  3. Estos elementos le deben siflexibilidad a su elemento padre
  4. Tenemos dos ejes: 1-Los elementos se despliegan de izquierda a derecha y transversalmente de arriba a abajo
  5. Todo se organiza por filas o columnas NO AMBAS!
